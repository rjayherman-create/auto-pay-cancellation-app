# Voice Over Studio - Pricing Strategy

## Executive Summary

**Goal**: Build sustainable, profitable business while keeping MVP free to build user base

**Pricing Model**: Freemium with paid tiers
**Target**: Break even at 50 paid users @ $29/month

---

## Tier Structure

### Tier 1: FREE (Entry) ⭐
**Price**: $0/month

**Includes**:
- Unlimited voice generation (with usage limits)
- 5 projects max
- Basic audio mixer
- Sound effects library
- 100 MB storage
- Community support

**Limitations**:
- 10 voice generations/month
- Watermark on exports (optional)
- No advanced features
- Community Discord only

**Target**: Students, hobbyists, content creators just starting

**Why Free?**
- Build user base quickly
- Network effects
- Convert to paid over time
- Get feedback for product improvement

---

### Tier 2: CREATOR ($9/month) 🎯
**Price**: $9/month (Annual: $79)

**Includes**:
- Unlimited voice generation
- 50 projects max
- Advanced audio mixer
- All sound effects
- 5 GB storage
- Email support
- Commercial use rights
- Export quality: 320kbps MP3

**New Features**:
- Voice blending (when available)
- Emotion variants
- Batch processing (generate 5 at once)
- API access (100 requests/day)

**Target**: Professional content creators, podcasters, YouTubers

**Conversion**: Upgrade from free when they hit 5 projects

**Why $9?**
- Affordable for individuals
- Covers ElevenLabs cost (~$5-10/month)
- Leaves room for hosting + margin
- Easy impulse buy

---

### Tier 3: STUDIO ($29/month) 💼
**Price**: $29/month (Annual: $249)

**Includes**:
- Everything in Creator +
- Unlimited projects
- Unlimited voice generations
- Advanced features
- 100 GB storage
- Priority email support
- Team collaboration (3 users)
- API access (1000 requests/day)

**New Features** (When Available):
- Music generator (Suno integration)
- Video studio
- Commercial generator
- 30-day free trial

**Target**: Small agencies, studios, production companies

**Why $29?**
- Covers ElevenLabs + Suno + infrastructure
- Leaves 40% margin for business
- Comparable to Adobe Creative Cloud pricing
- Professional tool pricing anchor

---

### Tier 4: ENTERPRISE (Custom) 🏢
**Price**: Custom (Starting $99/month)

**Includes**:
- Everything in Studio +

- Custom integrations

## Payment Processing

### Recommended: Stripe
- Setup: 5 minutes
- Cost: 2.9% + $0.30 per transaction
- At $9/month: costs ~$0.56/transaction
- Excellent for SaaS

### Setup:
```bash
npm install stripe

# Add to Node.js backend
const stripe = require('stripe')(process.env.STRIPE_SECRET_KEY);

app.post('/api/subscribe', async (req, res) => {
  const session = await stripe.checkout.sessions.create({
    payment_method_types: ['card'],
    line_items: [
      {
        price_data: {
          currency: 'usd',
          product_data: {
            name: 'Creator Monthly',
          },
          unit_amount: 900, // $9.00
          recurring: {
            interval: 'month',
          },
        },
        quantity: 1,
      },
    ],
    mode: 'subscription',
    success_url: `${process.env.DOMAIN}/success`,
    cancel_url: `${process.env.DOMAIN}/pricing`,
  });

  res.json({ sessionId: session.id });
});
```
