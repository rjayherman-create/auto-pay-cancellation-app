## 📞 Troubleshooting

### If Backend Stops
```
1. Check port 5000 is available
2. Check no other process using port
3. Manually restart: npm run backend:dev
```

### If Frontend Stops
```
1. Check port 8080 is available
2. Check Vite is running
3. Manually restart: npm run frontend:dev
```

### If Connection Issues
```
1. Verify both servers running
2. Check API proxy configuration
3. Check browser console (F12)
4. Verify ports 5000 & 8080 open
```
