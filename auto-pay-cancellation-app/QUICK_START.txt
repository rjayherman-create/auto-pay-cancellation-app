🔧 TROUBLESHOOTING

Services not running?
  → docker compose restart

Backend API not working?
  → curl http://localhost:8000/health
  → docker logs cardhugs-backend

Page not loading?
  → docker logs cardhugs-frontend
  → docker compose restart

Want to reset database?
  → docker compose down -v
  → docker compose up -d
