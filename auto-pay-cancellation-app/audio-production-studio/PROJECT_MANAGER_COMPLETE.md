## 🎯 PROJECT WORKFLOW

### Step 1: Create Project
1. Click **📁 Projects** in main menu
2. Click **➕ New Project** button
3. Enter project name (required)
4. Add description (optional)
5. Set duration in seconds
6. Click **✅ Create Project**

### Step 2: Add Project Details
1. Click **Open** on project card
2. Go to Project Details view
3. Add voices from Voice Library
4. Assign character names
5. Create scripts

### Step 3: Mix Audio
1. In project details
2. Click **🎚️ Open Mixing Board**
3. Mix all audio elements
4. Apply EQ & effects
5. Set levels

### Step 4: Export
1. Go back to project details
2. Click **💾 Export** buttons
3. Choose WAV or MP3 format
4. Download files
