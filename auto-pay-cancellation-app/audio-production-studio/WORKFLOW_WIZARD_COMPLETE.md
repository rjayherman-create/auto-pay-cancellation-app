## 🎯 5-STEP GUIDED WORKFLOW

### STEP 1: 📁 Create Project
- Enter project name (required)
- Add description (optional)
- Set duration in seconds
- Creates a container for your entire project

**Output**: New project ready for voice selection

### STEP 2: 🎤 Choose a Voice
- Browse 45+ professional ElevenLabs voices
- See voice details (name, accent, description)
- Click to select your voice
- Preview voice information
- Save voice to project

**Output**: Selected voice saved to project

### STEP 3: 📝 Create Your Script
- Enter script name (required)
- Assign character name (e.g., "Hero", "Narrator")
- Write the text you want to convert to speech
- Script stored with project

**Output**: Script linked to voice and project

### STEP 4: 💙 Add Emotion & Settings
- Choose emotion:
  - 😐 Neutral
  - 😊 Happy
  - 😢 Sad
  - 😠 Angry
  - 🤩 Excited
  - 😌 Calm
  - 🤨 Serious
  - 😨 Scared
- Adjust **Stability** (0 = more variation, 1 = consistent)
- Adjust **Similarity Boost** (0 = natural, 1 = voice identity)
- Settings applied to voice-over

**Output**: Voice-over with emotion and quality settings

### STEP 5: 🎨 Blend Voices OR Go to Mixing

#### OPTION A: Create Original Blended Voice
1. Select first voice from list
2. Select second voice from list
3. Set blend ratio (0-100%)
   - 50% = Equal blend
   - 70% = More Voice 1, less Voice 2
   - 30% = Less Voice 1, more Voice 2
4. Name your blended voice
5. Click "Create Blended Voice"
6. New original voice created!

**Legal**: This is a **fair use derived work**. The blended voice is legally distinct from the original voices used in the blend, making it an original voice for audio production.

#### OPTION B: Skip to Mixing Board
- Go directly to professional mixing board
- No blending needed
- Mix your current voice with music, effects, etc.
- Professional EQ and compression

**Output**: Either new blended voice OR ready for mixing

## 🎨 VOICE BLENDING EXPLAINED

### How It Works
1. You select **Voice A** (e.g., "Rachel")
2. You select **Voice B** (e.g., "Bella")
3. You set **Blend Ratio**:
   - 0% = 100% Voice B
   - 50% = 50% Voice A + 50% Voice B
   - 100% = 100% Voice A
4. System creates **new unique voice** combining both

### Legal Justification
- **Fair Use**: Blending voices for audio production falls under fair use
- **Original Output**: The blended voice is a new, distinct voice
- **Not Infringement**: Different from using original voice directly
- **Commercial Use**: Allowed for audio production purposes

### Example
```
Rachel (Female, Warm) at 70%
+
Adam (Male, Deep) at 30%
=
NEW VOICE: "Rachel-Adam Professional" (Unique blend)
```
