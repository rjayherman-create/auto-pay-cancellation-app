## 🔧 API ENDPOINTS (9 total)

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/available` | GET | List all effects |
| `/category/:cat` | GET | Effects by category |
| `/search?q=` | GET | Search effects |
| `/generate` | POST | Generate effect |
| `/save` | POST | Save to library |
| `/library` | GET | Get saved effects |
| `/library/:id` | GET | Get specific effect |
| `/library/:id` | PUT | Update effect |
| `/library/:id` | DELETE | Delete effect |
| `/library/:id/use` | POST | Record usage |
| `/layer` | POST | Layer effects |
| `/stats` | GET | Get statistics |
