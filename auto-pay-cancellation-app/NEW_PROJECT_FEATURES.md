## 🎯 QUICK START

### Create Project
1. Click "+ New Project"
2. Enter name & description
3. Click "Create Project"

### Add Voices
1. Open project
2. Select voice from dropdown (100+ available)
3. Enter character name (e.g., "Hero", "Villain")
4. Click "Add Voice"

### Manage Scripts
1. In project, click "Manage Scripts"
2. Create script for specific voice
3. Add tags, notes, status
4. Done!

### Generate Audio
1. Go to "🎤 Voice Generator" tab
2. Select voice from your project
3. Enter text
4. Click "Generate Voice"
5. Download!
