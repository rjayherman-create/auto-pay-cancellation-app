## 🎬 TYPICAL WORKFLOW

### Create 3-Minute Professional Video

```
Step 1: Open Video Studio (default)
        └─ Click 🎥 (already selected)

Step 2: Add Animation
        └─ Click 📁 Library
        └─ Select animation
        └─ Click + Add

Step 3: Add Voice-Over
        └─ Click + Add Track → Voice
        └─ Select or create voiceover
        └─ Set volume to 80%

Step 4: Add Background Music
        └─ Click + Add Track → Music
        └─ Select music
        └─ Set volume to 40%

Step 5: Add Sound Effects
        └─ Click + Add SFX at key frames
        └─ Add multiple effects as needed

Step 6: Add Text/Captions
        └─ Click + Add Text Track
        └─ Add captions with timing

Step 7: Preview & Adjust
        └─ Click ▶ Play
        └─ Listen to complete mix
        └─ Adjust timing and levels

Step 8: Export
        └─ Click 💾 Export
        └─ Choose MP4, High quality, 1080p
        └─ Click Start Export
        └─ Download MP4 file (30-60 seconds)

Total: ~7-10 minutes from start to export
```

## 💡 TIPS FOR BEST RESULTS

1. **Start with Video Studio**
   - All tools available in one editor
   - Default tab when app opens
   - Most intuitive workflow

2. **Use Animation Library**
   - Pre-made animations available
   - Quick to add
   - Professional quality

3. **Balance Audio Levels**
   - Voice: 80%
   - Music: 40%
   - Effects: 10-20%

4. **Add Captions**
   - Improves accessibility
   - Better engagement
   - Professional appearance

5. **Preview Before Export**
   - Always test timing
   - Check audio sync
   - Verify text placement

6. **Choose Right Quality**
   - YouTube: High (1080p)
   - Web: Medium (720p)
   - Preview: Low (quick)
