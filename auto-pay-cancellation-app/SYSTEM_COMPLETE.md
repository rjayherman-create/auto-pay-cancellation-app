

## 🎬 Workflow Example

### Create 3-Minute Animation

```
1️⃣  Upload Assets (2 min)
    └─ Background (1920×1080)
    └─ Character (512×512)
    └─ Audio (MP3/WAV)

2️⃣  Build Timeline (5 min)
    └─ Add keyframes at frames 0, 60, 120, etc.
    └─ Adjust position, scale, rotation, opacity
    └─ Click waveform to sync with audio peaks

3️⃣  Add Effects (1 min)
    └─ Select effect type
    └─ Set duration & parameters
    └─ Preview animation

4️⃣  Export Video (1 min)
    └─ Click "Export Video"
    └─ Wait for rendering
    └─ Download MP4 file

Total time: ~10 minutes per animation
```
