## 🎛️ Customize Everything

Every effect has 5 adjustable parameters:

- **Duration** (0.1 - 3 seconds)
- **Pitch** (0.5x - 2x)
- **Intensity** (0.5x - 2x)
- **Reverb** (0 - 1)
- **Speed** (0.5x - 2x)

**Same effect, different parameters = completely new sound**

## 🎓 Pro Tips

✅ **Pitch is powerful** - Change pitch = new sound  
✅ **Layer effects** - Combine multiple for richer audio  
✅ **Save variations** - Generate 3 versions, pick best  
✅ **Mix with music** - Always add background layer  
✅ **Extend duration** - Same effect, longer play = different feel  
✅ **Increase intensity** - More emphasis for key moments
