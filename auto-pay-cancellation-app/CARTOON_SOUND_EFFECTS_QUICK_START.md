## 🎛️ Customize Parameters

**All Effects Have:**
- Duration (0.1 - 3 seconds)
- Pitch (0.5x - 2x)
- Intensity (0.5x - 2x)
- Reverb (0 - 1)
- Speed (0.5x - 2x)

**Example Customization:**
```
Original: Jump sound
Pitch: 1.3x (higher)
Intensity: 1.2x (louder)
Speed: 1.0x (normal)

Result: High-pitched, emphasized jump
```

## 🎯 Common Tasks

### Create Chase Scene Audio
```
1. Generate: Jump (pitch 1.3x)
2. Generate: Running (speed 1.5x)
3. Generate: Whoosh (intensity 1.2x)
4. Generate: Crash (duration 0.8s)
5. Mix all 4 in Audio Mixer
6. Download combined track
```

### Create Magic Effect
```
1. Generate: Transformation (reverb 0.8)
2. Generate: Sparkle (pitch 1.8x)

5. Export magical sound
```

## 💡 Pro Tips

✅ **Pitch makes big difference** - Same effect, different pitch = new sound  
✅ **Intensity adds emphasis** - Higher = more impact  
✅ **Duration extends** - Short snappy vs long sustained  
✅ **Speed changes feel** - Fast = comedic, Slow = dramatic  
✅ **Layer effects** - Multiple effects = richer sound  
✅ **Save variations** - Generate 3 versions, pick best  
✅ **Mix with music** - Always add background music layer
