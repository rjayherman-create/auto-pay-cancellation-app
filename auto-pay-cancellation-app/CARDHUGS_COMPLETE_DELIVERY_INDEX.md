## 🔌 API Endpoints

### LoRA Training
```
POST   /api/training                Create training job
GET    /api/training                List all jobs
GET    /api/training/:id            Get job status
DELETE /api/training/:id            Delete job
```

### Card Creator
```
POST   /api/card-creator/generate-text     Generate text
POST   /api/card-creator/generate-image    Generate image
POST   /api/card-creator/generate-element  Generate element
POST   /api/cards/save-complete            Save card
```

### AI Titles
```
POST   /api/ai/generate-title              Generate title
POST   /api/ai/generate-titles             Generate variations
POST   /api/ai/check-title                 Check duplicate
GET    /api/ai/title-stats                 Get statistics
```
